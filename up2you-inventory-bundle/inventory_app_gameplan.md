# Jewelry Inventory Web App - Game Plan

**Domain:** inventory.kvn.ltd (or your-decide.kvn.ltd)  
**Tech Stack:** React (frontend) + Node/Express (backend) + SQLite (database)

---

## Phase 1: Setup & Infrastructure
- [ ] Create subdomain on kvn.ltd pointing to your server
- [ ] Set up Node.js project structure on server
- [ ] Install dependencies (Express, SQLite3, CORS, etc.)
- [ ] Create empty SQLite database file

## Phase 2: Backend API (Node/Express)
- [ ] POST /api/items - Add inventory item
- [ ] GET /api/items - Fetch all inventory
- [ ] PUT /api/items/:id - Update item
- [ ] DELETE /api/items/:id - Delete item
- [ ] GET /api/export/csv - Export as CSV
- [ ] GET /api/export/labels - Generate label data for printing

## Phase 3: React Frontend
- [ ] Inventory form with guided prompts
  - Item name, description, materials, price, quantity, photo
  - Category/tags
  - Storage location
- [ ] Dashboard view (inventory table, counts, totals)
- [ ] Export buttons (CSV, labels, listing templates)
- [ ] Edit/delete functionality
- [ ] Basic styling (clean, functional)

## Phase 4: Polish & Testing
- [ ] Photo upload handling
- [ ] Error handling
- [ ] Mobile-friendly UI
- [ ] Test with sample data
